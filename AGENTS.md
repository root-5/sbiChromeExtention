# リポジトリガイドライン

## プロジェクト概要

このリポジトリは SBI 証券の HP で動作する Google Chrome 拡張機能を開発するためのものです。
具体的には、ポートフォリオ、取引履歴、価格推移などを取得し、優れた UI で表示することを目的としています。
Manifest V3 に準拠し、サービスワーカーを使用してバックグラウンドで処理を実行します。

## コーディング規約

- コメントやドキュメントは日本語で記載すること。また、コメントはかなり丁寧に書くこと
- 既存のコードスタイルに従うこと
- ファイル名、変数名、関数名、クラス名はキャメルケースを使用すること
- 過剰なエラー処理を避け、必要最低限に留めること

## メンテナンス_ポリシー

- 冗長だったり、圧縮の余地がある箇所を検討すること
- リファクタリングを行える箇所があったときは提案を行うこと
